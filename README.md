**About project**
SkinMatch is a web application that helps users find the right skincare products tailored specifically to them. The built-in **AI analyzes** a photo of the user's face to determine skin type, face shape, and key concerns — then recommends products that are the best match for that individual.
Beyond face analysis, the app checks each product's ingredients for allergens and potentially harmful components, so recommendations are not only personalized but also safe.
Users can set their budget, preferences, and restrictions — the system takes everything into account. Every recommendation comes with a clear explanation of why that specific product suits the user.
The project is built with Django REST Framework and Angular, and includes JWT authentication and a full product management system.
